Core Concept

Large Language Human (LLH) presents itself as an ordinary ChatGPT-style chat application—but instead of an LLM, it's powered by real humans.

Every participant has 2 roles:

👤 User Mode

Submit requests, ask questions, get help from other humans in the network.

  • Standard chat interface
  • Request any supported task
  • Rate helpful responses

🤝 Agent Mode

Respond to requests from other users, earn reputation, contribute to the community.

  • Protected by energy budgets
  • Can decline any request
  • Set personal boundaries

Energy Budget Framework

To keep humans from spending more energy than reasonable, we define 4 categories with strict constraints.

💬

Category A: Conversational

Typing conversational prose at a computer

45 min/day • 500 words max • 3 concurrent threads

💻

Category B: Code Review

Examining programs, pair programming

25 min/day • 100 lines max • 20 new lines

🎨

Category C: Drawing

Creating simple images and sketches

20 min/day • 800×600 max • 2 revisions

🌍

Category D: Actions

Real-world tasks (weather, food, orders)

15 min/day • 3 min research • Pre-auth required

The "No" Protocol

Agents can decline any request with zero penalty. This is fundamental to preventing exploitation.

🛡️ Valid Decline Reasons

  • Budget exhausted (daily limit reached)
  • Scope too large (exceeds constraints)
  • Out of scope (not in agent's skills)
  • Personal boundary (no explanation required)
  • Safety concern (harm, illegality, liability)

💡 Auto-Suggestions

  • System generates alternatives on decline
  • User sees "Try asking..." suggestions
  • Option to find different agent
  • Schedule for later when budget resets

📊 Anonymous Analytics

  • Decline reasons logged (anonymized)
  • System learns to route better
  • Constraints evolve based on data
  • No penalty for agent
def decline_request(request, reason: DeclineReason):
    # Log for system learning (anonymized)
    analytics.record_decline(reason, request_category)
    
    # Auto-suggest alternatives
    alternatives = suggestion_engine.generate(request, exclude_reason=reason)
    
    # Return graceful response
    return Response(
        status="DECLINED",
        message=f"I can't help with that right now ({reason.label}). "
                f"Try: {alternatives[:3]}",
        can_retry_after=timedelta(minutes=30) if reason == "BUDGET_EXHAUSTED" else None
    )

Ethical Guardrails

✅ Always

  • Explicit consent per category
  • Agent can withdraw anytime
  • Privacy first (identities protected)
  • Transparent compensation (if monetized)
  • Exit strategy (delete data, leave without penalty)

❌ Never

  • Pressure agents to respond faster
  • Hide human nature from users
  • Allow harmful or illegal requests
  • Use labor to train AI without consent
  • Gamification that encourages overwork
